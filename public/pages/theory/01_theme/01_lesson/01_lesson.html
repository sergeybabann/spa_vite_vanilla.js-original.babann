Урок 01: Введение

Для кого этот курс?

Язык программирования JavaScript позволяет работать не только для frontend, но и для backend. Подготовлено более 300 слайдов с описанием языка и его возможностей, а также многими примерами.

Этот курс подойдет для начинающих. Также для тех кто пытался выучить этот язык, но по каким-то причинам не смог. Для тех кто имел опыт работы с языком, но хотел бы глубже разобраться с языком программирования. Для тех кто просмотрел десятки видео по языку, но не может написать ни одной программы.

Как проходить этот курс?

Писать код самостоятельно

Запускать код, модифицировать его и снова запускать пока на 100% не будет понятно что происходит при запуске

Когда перестает быть понятно, возвращайтесь назад и пересматривайте предыдущие части

Используйте Notion или другие программы для записи ключевых понятий и объяснений в своей интерпретации

Как НЕ надо проходить курс

Нельзя просто просматривать информацию и ничего не делать. Это бесполезно потраченное время. Обязательно нужна практика, нужно писать код и видеть результат.

Что изучать после курса?

Изучать библиотеки и фреймворки для создания веб-приложений, такие как React, Vue.js, Angular для создания фронтенд приложений

Изучать React Native для создания мобильных приложений для iOS и Android

Изучать Node.js для создания Backend приложений

Принцип 80 / 20

Это принцип ксается не только языка JavaScript, он распространяется на многие аспекты различных сфер. В частности если говорить про JavaScript, то к числу 80 относится великое множество возможностей языка, и эту величину отнесем к процентам. 80% это то количество, которое существует в языке, но применяется довольно редко. И 20% его составляющей используется постоянно.

Где выполнять JavaScript код?

JavaScript код можно выполнять в нескольких средах:

  • Веб-браузер. JavaScript встроен в любой веб-браузер
  • Node.js
  • В любой среде разработки IDE. Например Visual Studio Code
