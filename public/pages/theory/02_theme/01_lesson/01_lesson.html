Урок 01: Самое важное в JavaScript

Самое важное в JavaScript это - выражения, функции, объекты

Главная идея в JavaScript в том, что практически все сущности в JavaScript это - объекты. Объект это - набор свойств, имя:значение, либо пара имя:значение. Имена еще называют ключами.

Пример объекта : 

                
                // example object

    {
        visible: true,
        colorDepth: 24,
        title: 'My image',
        Orientation: {
            angle: 0,
            type: 'landscape'
        },
        pixelDepth: 24,
        width: 1440
    }

visible colorDepth title Orientation -- это все свойства конкретного объекта.

true 24 'My image' и т.д. -- это значения соответствующих свойств.

У некоторых свойств значением может быть также объект. Такие объекты называются вложенными объектами.

В данном случае вложенным объектом является: 
                
            Orientation: {
                angle: 0,
                type: 'landscape'
            },

У этого вложенного объекта есть свои свойства со значениями.

В JavaScript :

  • Массив это - объект
  • Функция это - объект
  • Число это - объект
  • Строка это - объект

Числа и строки только ведут себя как объекты. Значения такие как число и строка это примитивные значения. Все сущности в JavaScript - это объекты. Объект это набор свойств, и у свойств есть свои значения.
